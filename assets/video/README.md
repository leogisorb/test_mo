# Video Assets

Platzieren Sie hier das Hero-Video für die Startseite.

## Anforderungen:
- **Dateiname**: `diving-hero.mp4` (oder `.webm`)
- **Format**: MP4 oder WebM
- **Auflösung**: Mindestens 1920x1080 (Full HD)
- **Eigenschaften**:
  - Automatisches Abspielen
  - Stumm (ohne Ton)
  - Endlosschleife
  - Sollte Tauchen, Meer, Unterwasserwelt zeigen

## Empfohlene Video-Eigenschaften:
- Codec: H.264 (MP4) oder VP9 (WebM)
- Bitrate: ~5-10 Mbps
- Dauer: 30-60 Sekunden (wird in Endlosschleife abgespielt)
