# Bild Assets

Platzieren Sie hier die Bilder für das Social Proof / Instagram Grid.

## Anforderungen:
- **Dateinamen**: `dive-1.jpg`, `dive-2.jpg`, `dive-3.jpg`, `dive-4.jpg`, `dive-5.jpg`, `dive-6.jpg`
- **Format**: JPG oder PNG
- **Größe**: Quadratisch empfohlen (z.B. 800x800px oder 1000x1000px)
- **Inhalt**: Tauchbilder, Unterwasserwelt, Taucher, Korallenriffe, etc.

## Empfohlene Bild-Eigenschaften:
- Seitenverhältnis: 1:1 (quadratisch)
- Auflösung: Mindestens 500x500px
- Dateigröße: Optimiert für Web (unter 500KB pro Bild)
- Format: JPG für Fotos, PNG für Grafiken mit Transparenz

Die Bilder werden automatisch in einem responsiven Grid angezeigt und passen sich der Bildschirmgröße an.
